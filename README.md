# MA1805-Spring-Project-

https://tidiane077.github.io/MA1805-Spring-Project-/

Group Members: Tidiane Barry(Coder + Research) & Mia Brock(Coder + Designer) 

Project Description: A demo game called 'SPACE RIDER' which involves an Alien riding a hoverboard in space trying to destroy an asteroid belt that is coming towards its planet. PLayers must aim and tap the screen to shoot the asteroids before they go past the space rider. If 5 go past the space rider, the game will end and his planet will be destroyed. 

Made use of sketch.js to develop game mechanics, css to solidofy the astethetic and finally html to bring it all together.

YT tutorials like coding with Mr Ward and The coding Train helped to develop shooting mechanics along with other movements.
